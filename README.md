## Run locally

1. Create python virtual environment and activate it:
   
   ```
   python -m venv ./venv
   .\venv\Scripts\activate
   ```

3. Install packages:

   ```
   pip install -r requirements.txt
   ```

4. Install playwright's dependencies:

   ```
   playwright install
   ```

5. Start script:
   ```
   python main.py
   ```
